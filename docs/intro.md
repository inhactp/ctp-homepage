# CTP 소개

CTP(Challenge The Programming)는 자료구조 및 다양한 알고리즘을 학습, 연구하거나 여러 프로그래밍 대회를 준비하는 학술 소모임입니다.  
기초적인 수학 문제부터 어려운 알고리즘이 필요한 문제까지 다양한 문제들을 풀면서 수학적 사고력과 문제 해결 능력을 기르는 것을 목표로 하고 있습니다.
